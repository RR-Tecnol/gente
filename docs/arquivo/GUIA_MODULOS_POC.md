## #01 — Portal do Servidor — Dashboard | ÁREA DO SERVIDOR

*Painel inicial personalizado que centraliza o acesso a todos os módulos do sistema, com notificações em tempo real e indicadores do período atual.*

**Funcionalidades**
- Painel personalizado com nome, cargo e setor do servidor
- Acesso rápido a todos os módulos disponíveis
- Notificações em tempo real: pendências, comunicados e vencimentos
- Indicadores: abonos pendentes, status da folha e competência atual
- Comunicados internos por categoria (RH, Financeiro, Operacional, Urgente)
- Agenda com eventos, reuniões e plantões programados

| | |
|---|---|
| **Perfis de Acesso** | Servidor |
| **Integrações** | Recebe dados de todos os módulos do sistema. |
| **Destaque** | **Ponto de entrada único para o servidor — tudo em uma tela.** |
